### Overview

**Part I: C# Fundamentals** introduces you to the essential concepts, syntax, and paradigms of the C# language.  It begins by exploring the differences between .NET Core and .NET Framework.  This course then provides a comprehensive overview of C#â€™s features and how it is used to create object-oriented applications.

**Part II: Advanced C#** lets you explore the power of C#. Dive deep into the intricacies of data handling, from mastering value vs reference types and memory allocation to working with dynamic and nullable types. Gain proficiency in managing unmanaged resources with the `IDisposable` and finalizers. Enhance your coding prowess with advanced topics such as delegates, asynchronous programming using TPL, and threading. Explore the power of C# reflection for dynamic typing and late binding, and understand the essentials of encryption.


### Duration
* **Part I**: 4 days
* **Part II**: 3 days

### Objectives
Upon completion of this course, you will be able to:
* Build a C# applications using essential syntax and flow control
* Apply object-oriented programming principles like encapsulation and inheritance
* Work with data using collections, generics, and file I/O operations
* Write modular, maintainable code and handle errors effectively
* Manage resources to avoid memory leaks and ensure efficient cleanup
* Understand delegates and their use in the .NET libraries
* Write asynchronous code with keywords `async` and `await` 


### Setup
All you really need is the .NET sdk and a text editor. Everything in this repository uses [.NET 9](https://dotnet.microsoft.com/en-us/download/dotnet/9.0), but most projects will compile with earlier versions.
