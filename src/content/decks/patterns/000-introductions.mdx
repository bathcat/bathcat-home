### On Labs

<figure class='bc-framed-image' style='width:30%;margin-left:1rem;'>

> [Learning design patterns] is like basketball: it takes a lot of practice to do well.  
>  --G. Stolfus

- Learn design patterns by coding up implementations
- Advantages
  - You’ll remember more
  - Some patterns can be tricky to implement
- Disadvantages
  - You might go pattern crazy
  - You’ll still have to look up implementations online
