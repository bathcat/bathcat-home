### Overview

<figure class='bc-framed-image' style='width:20%;margin-left:1rem;transform: scaleX(-1);float:right;'>
![Image sans description.](assets/snover.jpeg)
</figure>

- Replacement for `Cmd.exe`
- Originally known as [Monad](https://www.jsnover.com/Docs/MonadManifesto.pdf)
- Architect: [Jeffrey P. Snover](https://en.wikipedia.org/wiki/Jeffrey_Snover) [Twitter](https://twitter.com/jsnover)
- First release: 2007

---

### Windows PowerShell

- .NET-Based
- Broad support
- Does everything

```powershell
Add-Type -TypeDefinition @"
public static class Mathy{
  public static int Add(int a, int b){
    return (a + b);
  }
}
"@
[Mathy]::Add(5,19)
```

- Cross platform

## Kudos

---

### Powerful

- Entire .NET platform + ecosystem
- e.g. Fancy find-and-replace script

```powershell

function fix-slidebreaks($path){
  (Get-Content $path -Raw) -replace '(?ms)\s^(?=#{1,3}\s)', "---`n`n"
    | Out-File $path
}

Get-Item ./src/content/ |
   Get-ChildItem -Recurse -Filter content.mdx |
   %{fix-slidebreaks $_}

```

---

### Ergonomic

- Verb-noun makes it easy to read
  - Unlike `dir`, `cat`, `set`, `grep`
- Help system
- Tab completion
- Great tooling
  - [Windows terminal](https://github.com/microsoft/terminal)
  - [VSCode](https://code.visualstudio.com/)

---

### Useful

- Automates boring stuff

```powershell

npm run ssr
cd dist
# By default, gihub pages ignores folders prefixed with underscores.
echo '' >> .nojekyll

# This is so the custom domain setting won't get clobbered
echo 'bathcat.net' >> CNAME

git init
git add -A
git commit -m 'deploy'

git push -f https://github.com/bathcat/bathcat.github.io +master
```

### Platforms Aren't the same

- Lots of functionality is Windows-only
  - [`Get-ComputerInfo`](https://docs.microsoft.com/en-us/powershell/module/microsoft.powershell.management/get-computerinfo)
  - [`Get-Service`](https://learn.microsoft.com/en-us/powershell/module/Microsoft.PowerShell.Management/Get-Service)
  - WMI/CIM Stuff
  - Scheduled jobs
  - Remoting
- Commands on Windows: **~1600**
- Commands on Ubuntu: **~350**
